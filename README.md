# Environment Configuration

Below are the environment variables required to run this project.  
Create a `.env` file and add the following values:

```env
PYMONGO_URI=""
PYMONGO_DB=""

VITE_FRONTEND_BASE_URL=""

SECRET=""
TOKEN_EXPIRY_TIME=15

REDIS_HOST=""
REDIS_PORT=11502
REDIS_USERNAME="default"
REDIS_PASSWORD=""

SMTP_SERVER="smtp.gmail.com"
SMTP_PORT=587
SMTP_PASSWORD=""
